# django2 - Projeto genérico utilizando MySql e envio de emails

## Pré-requisitos

Para conseguir implementar este software são necessarios os seguintes itens:

```text
Python 3.12 ou superior
Pip 20.0 ou superior
virtualenv 20.4 ou superior
django 4.2.6
```

## Instalação

Instalar ambiente virtual no diretório do projeto.

```text
python -m venv myenv
```

Ativar o ambiente virtual.

No windows

```text
myenv\Scripts\activate.bat
```

No Unix

```text
source myenv/bin/activate
```

Instale as dependências do Projeto no arquivo requirements.txt

```text
pip install -r requirements.txt
```

Para cada nova dependência, atualize o requirements.txt

```text
pip freeze > requirements.txt
```
